# Tech Stack Explorer

## Installation et démarrage

1. Clone le repo : git clone https://github.com/Doumbouya967
2. Navigue dans le dossier : cd tech-stack-explorer
3. Installe les dépendances : npm install
4. Lance le serveur de dev : npm run dev

## Éditer le code

- Ouvre le projet dans ton éditeur (ex. VS Code).
- Modifie les fichiers dans src/.
- Teste localement avec npm run dev.

## Technologies utilisées
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase
